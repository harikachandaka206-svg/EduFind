# EduFind

EduFind is a machine learning-based web application that recommends educational resources based on a student's subject, topic, and difficulty level.

## Features

- Resource recommendation
- Machine Learning model
- Flask web application
- Easy-to-use interface

## Technologies Used

- Python
- Flask
- Pandas
- Scikit-learn
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
