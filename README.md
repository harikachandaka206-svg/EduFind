# 🎓 EduFind – Educational Resource Recommendation System

## 📌 Overview

EduFind is a Machine Learning-based web application that helps students discover relevant educational resources based on their selected subject, topic, and difficulty level. The application provides personalized recommendations through a simple and user-friendly web interface built with Flask.

---

## ✨ Features

* 📚 Recommends learning resources based on user input
* 🤖 Machine Learning-powered recommendation system
* 🌐 Interactive web interface using Flask
* 📊 Uses a dataset of educational resources
* ⚡ Fast and easy-to-use search functionality

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/harikachandaka206-svg/EduFind.git
```

Move into the project folder:

```bash
cd EduFind
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

Start the Flask application:

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000/
```
